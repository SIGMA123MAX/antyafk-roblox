--------------------------------------
  REQUIREMENTS
--------------------------------------
  - Windows 10 or 11
  - AutoHotkey v1.1 (NOT v2.0)
    Download: https://www.autohotkey.com
    -> Choose "Current Version" (v1.1.x)

--------------------------------------
  HOW TO RUN
--------------------------------------
  1. Install AutoHotkey v1.1
  2. Double-click antifk_pro_v3_kppi3.ahk
  3. The GUI will open
  4. Set your interval + jump style
  5. (Optional) paste Discord webhook URL
  6. Press START

--------------------------------------
  FEATURES
--------------------------------------
  - Auto-detects ALL Roblox windows
  - Jump variants: 1/2/3 jumps, rotate,
    crouch, random mix
  - Randomize + anti-pattern delay
  - Send chat message after jump
  - Discord webhook with custom embed fields
  - Screenshot capture per window
  - Attach screenshots to Discord
  - Session timer + jump counter
  - Start notification to Discord
  - System tray support

--------------------------------------
  DISCORD WEBHOOK SETUP
--------------------------------------
  1. Open your Discord server
  2. Channel Settings -> Integrations
  3. Webhooks -> New Webhook
  4. Copy URL -> paste into the app

--------------------------------------
  HOTKEYS
--------------------------------------
  Win + Esc = Exit the script

--------------------------------------
  NOTE ABOUT ANTIVIRUS
--------------------------------------
  AHK scripts that simulate keypresses
  may trigger false positives. The code
  is fully open - read it in Notepad.
